Origins

From humble beginnings, Quizard has been developed to be a fully self-contained flask application ready to be hosted on the web!

Originally, Quizard was created to fulfil the requirements for the second project of CITS5505: Agile Web Development. However, the two developers (Gerardo and Cameron) quickly realised that they were onto a winner. By utilising all the tools that web development has to offer

  • HTML5
  • CSS (Custom & Bootstrap)
  • JavaScript
  • Flask

They have developed Quizard to be one of the most functional open source quiz applications there is.

Features

Quizard is packed full of features, raring to go for any user!

Quizard has been design with user-friendliness at the fore front of every decision - designed by web users, for web users! Using Quizard, you can develop:

  • Multi-choice
  • Voting
  • Free-text
  • Any combination of the above

To your heart's content!
